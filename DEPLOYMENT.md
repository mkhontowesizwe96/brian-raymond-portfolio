# Deploying This Portfolio

## GitHub Pages

1. Create a public GitHub repository named `brian-raymond-portfolio` under your GitHub account.
2. Push this folder to the repository.
3. In GitHub, open the repository settings.
4. Go to `Pages`.
5. Set the source to `Deploy from a branch`.
6. Select the `main` branch and `/root`.
7. Save.

Your site will be available at:

`https://mkhontowesizwe96.github.io/brian-raymond-portfolio/`

## Custom Domain Later

If you buy a domain later, add it in the same GitHub Pages settings screen and update your DNS records as GitHub instructs.
